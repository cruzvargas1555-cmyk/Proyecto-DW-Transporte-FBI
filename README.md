# Proyecto-DW-Transporte-FBI

Proyecto final de Inteligencia de Negocios aplicando la metodología HEFESTO,
desarrollado para la empresa de transporte "FBI". Incluye el diseño e implementación
de un Data Warehouse, un Cubo OLAP (SSAS) y Dashboards interactivos en Power BI.

## 📂 Estructura del repositorio

- **SQL/** → Scripts de creación de tablas, carga de datos (ETL manual) y backups de la base de datos (OLTP y Data Warehouse).
- **SSIS/** → Proyecto de Integration Services, usado para el proceso ETL de carga automatizada.
- **SSAS/** → Proyecto del Cubo OLAP (Analysis Services), con sus dimensiones, vista de origen de datos y grupo de medidas.
- **PowerBI/** → Dashboards (.pbix) cada uno respondiendo preguntas de negocio y presentando KPIs.
- **Documentacion/logo/** → Recursos gráficos (logo de la empresa) usados en los dashboards.
